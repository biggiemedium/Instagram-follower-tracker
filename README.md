## Features

- **Drag & Drop Interface** - Simply drop your `followers.json` and `following.json` files
- **Relationship Analysis** - Find mutual follows, one-way relationships, and non-followers
- **Advanced Filtering** - Filter by relationship type, time range, and search usernames
- **Timeline View** - Visualize your follow activity over time
- **Suspicious Account Detection** - Identify potentially fake or spam accounts
- **Export Capabilities** - Export filtered data to CSV or detailed JSON reports
- **Privacy First** - All data processing happens locally in your browser

## Usage

1. Download your Instagram data from Instagram's "Download Your Information" feature
2. Extract the `followers.json` and `following.json` files from the download
3. Open `index.html` in your web browser
4. Drag and drop both JSON files into their respective zones
5. Explore your relationships with various filters and insights

## Analysis Types

- **Overview** - Mutual follows, one-way relationships summary
- **Relationships** - Detailed relationship breakdowns and growth insights
- **Activity** - Recent activity patterns and timeline analysis
- **Insights** - Inactive accounts and suspicious account detection
- **Timeline** - Visual timeline of your Instagram activity

## Files

- `index.html` - Main application interface
- `styles.css` - Professional dark theme styling
- `script.js` - Core analysis logic and UI functionality

## Privacy

Your Instagram data never leaves your device. All processing is done client-side in your browser with no server uploads or external API calls.
